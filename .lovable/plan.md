
# تحسين مودال نقل الرفوف - عرض الرفوف مع البضاعة

## ملخص
تعديل مودال نقل الرفوف ليعرض المنتجات مجمّعة حسب الرف بدلاً من قائمة مسطحة، مع إضافة البحث بأسم الرف لتسهيل العثور على البضاعة المطلوب نقلها.

## التغييرات

### تعديل `TransferShelfModal.tsx`

1. **تجميع المنتجات حسب الرف**: بدلاً من عرض قائمة مسطحة، يتم تجميع المنتجات تحت أقسام مسمّاة بأسم الرف (Accordion/Collapsible style) مع عدد المنتجات في كل رف.

2. **توسيع البحث ليشمل اسم الرف**: حقل البحث الحالي يبحث بالاسم والكود والباركود فقط. سيتم إضافة البحث بأسم الرف (`rafKonum`) بحيث عند كتابة اسم رف تظهر كل منتجاته.

3. **تحديد كل منتجات رف دفعة واحدة**: زر "تحديد الكل" بجانب كل رف لاختيار جميع منتجاته بضغطة واحدة.

4. **تصميم القسم الجديد**:
   - كل رف يظهر كعنوان قابل للطي (Collapsible) مع أيقونة MapPin وعدد المنتجات
   - زر "تحديد الكل" / "إلغاء الكل" بجانب اسم الرف
   - المنتجات تحته بنفس التصميم الحالي (Checkbox + اسم + كود)
